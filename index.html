Index is a market-cap-weighted composite (base = 1 Apr 2023, base value 1000), computed the same way most equity indices work: index = (Σ price × locked share count) ÷ divisor. Share counts are frozen at construction, not refetched every run, so day-to-day moves reflect price only — see the repo's establish_base.py to rebalance. Prices refresh every 15 min during NSE hours via scheduled GitHub Actions, whether or not anyone is viewing this page.
